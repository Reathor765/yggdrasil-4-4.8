# Yggdrasil

**Yggdrasil** is an editor tool and runtime scripting system for creating, editing, managing, and using node-based trees in games.

It is designed for structures such as **Passive Trees**, **Spell Trees**, and other graph- or node-driven progression systems, giving you both the tooling to build them and the runtime support to use them in-game.

## Features

- Registry that holds groups of trees
- Loader to easily return tree resouce
- Builder to turn tree resource into working tree with only few lines of code
- Versatile editor that allows fast and efficient tree creation
- Editor tabs for opening multiple trees at once
- Prefabs that speed up nodes reusability
- Grid snapping
- Shortcuts
- Validator to help catch less or more important errors

## Use Cases

- Passive skill trees
- Spell or ability upgrade trees
- Technology or research trees
- Quests or achievements progression
